# Crochet Globe Generator

This tool generates crochet patterns for spherical objects such as globes and pillows.

## Features

- Custom globe diameter
- Yarn consumption estimation
- Support for multiple colors

## Usage

1. Create a test circle.
2. Measure stitch dimensions.
3. Enter the values.
4. Click **Generate**.

## Abbreviations

- `sc` = single crochet
- `inc` = increase
- `dec` = decrease
